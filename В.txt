HTML - язык разметки гипертекста. Нужен, в основном, для создания документов в интернете. Разработан в 1993.

JavaScript - встраиваемый язык для программного доступа к объектам приложений. 
CSS - Cascading Style Sheets - Каскадные таблицы стилей. Язык описания внешнего документа.
Гипертекст - система из текстовых страниц, имеющих перекрёстные ссылки.
